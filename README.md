# Yash Chauhan | 3D Developer Portfolio

A premium, interactive 3D portfolio website built natively with HTML, CSS, and JS. This portfolio serves as a showcase of my journey as a Full Stack Developer & Cloud Enthusiast.

## 🌟 Key Features
- **Dark Premium Aesthetic:** A modern mix of deep space obsidian with glowing neon cyan and purple accents.
- **Scroll Reveal Animations:** Built completely from scratch using the `IntersectionObserver` API for smooth staggred upward fades.
- **3D Interactive Elements:** Projects and Technical Skills feature ReactBits and Framer Motion-inspired dynamic 3D tilt cards that react to mouse movement.
- **Animated Mesh Background:** A pure CSS-driven glowing blob mesh background that breathes life into the page.
- **Zero Dependencies:** Fully built natively without any heavy frontend frameworks (React, Vue, etc). This makes deployment instant and extremely lightweight.

## 🚀 Live Preview
You can deploy this instantly using GitHub Pages!
Simply go to your repository settings -> **Pages** -> Deploy from branch `main` to get your live URL.

## 💻 Local Setup
Since there are no build steps:
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Itschauhanyash/profile.git
   ```
2. Open `index.html` directly in your web browser. 
3. (Optional) Run a local server for a better development experience: `python -m http.server 8080`

## 🎨 Customization
All styling tokens such as theme colors, fonts, and base animation speeds are neatly centralized at the top of `style.css` inside the `:root` variables block for easy modification.
